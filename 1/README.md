Задание
=======
Реализовать функции ```(@NTHCDR n list )``` и ```(@NTH n list)``` , возвращающие соответственно n-й CDR и CAR от списка list. Некорректное задание параметра n должно контролироваться.

Примеры
=======
ЛИСП
```lisp
(@NTHCDR 0 '(А В С))
```
(A B C)

```lisp
(@NTHCDR 2 '(А В С D))
```
(C D)

```lisp
(@NTH 0 '(А В))
```
A

```lisp
(@NTH 1 '((АВ)(ВС)(CD)К))
```
(BC)
***
ПРОЛОГ
```prolog
NTHCDR(0, [1, 2, 3], L),
write(L).
```
[1, 2, 3] 

```prolog
NTHCDR(2, [1, 2, 3, 4], L),
write(L).
```
[3, 4]

```prolog
NTH(0, [1, 2], E),
write(E).
```
1

```prolog
NTH(3, [1, 2, 3, 4, 5, 6, 7], E),
write(E).
```
4
